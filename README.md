<h2>Развертывание</h2>
Для работы с программой сначала нужно развернуть базу данных. Скрипт по ее развертыванию находится в файле InitSqlScript.sql. Его нужно выполнить для создания базы данных, если она еще не создана. Для этого отлично подойдет приложение SQL Server Management Studio.

Само приложение запустить довольно просто - клонировать репозиторий, открыть файл решения .sln и собрать проект самостоятельно в Visual Studio 2022, все зависимости будут загружены пакетным менеджером NuGet автоматически и скомпилированная программа появится в выходной директории. Остается только поменять строку подключения к базе данных в файле App.config.

<h2>Особенности</h2>
<b>В программе имеются:</b>
<ol>
  <li>Удобная навигация</li>
  <li>Возможность смены таблицы стилей по кнопке справа сверху</li>
  <li>Отображение подсказок при наведении на элементы, где необходимо</li>
  <li>Отображение текущего времени в нижнем правом углу программы</li>
  <li>Система учетных записей</li>
  <li>Система смены пароля</li>
  <li>Система хеширования паролей</li>
  <li>Система капчи</li>
  <li>Таблицы пользователей, платежей и их категорий</li>
  <li>Отображение всех записей из таблиц базы данных в графическом интерфейсе, для пользователей в в виде плиток без возможности редактирования, для администраторов в виде таблицы с возможностью редактирования</li>

  <li>Отображение и установка изображений в качестве аватарок для выбранных пользователей</li>
  <li>Система отображения диаграмм выбранного типа для указанного пользователя по его платежам</li>

  <li>Продвинутая система обработки ошибок и исключений, которая сообщит пользователю, если что-то пойдет не так понятным сообщением</li>
